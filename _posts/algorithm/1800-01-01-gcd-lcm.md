## gcd 와 lcm

영어 약자로 최대공약수는 gcd, 최소공배수는 lcm 이라고 한다. gcd 와 lcm 을 구하는 함수에 대해 포스팅하고자 한다.

## 두 수의 gcd, lcm 구하는 함수

어떤 두 양수 x, y 가 있을 때, 이들의 gcd 를 구하는 가장 유명한 방법은 [유클리드 호제법](https://namu.wiki/w/%EC%9C%A0%ED%81%B4%EB%A6%AC%EB%93%9C%20%ED%98%B8%EC%A0%9C%EB%B2%95)이다. 무엇인지 잘 모르겠다면 링크 내용을 확인해보자. 그리고 lcm 은 x, y 를 곱한 뒤 두 수의 gcd 로 나누면 된다.

아래는 이를 함수로 구현한 코드다.

```py
def gcd(x, y):
    return gcd(y, x%y) if x%y else y

def lcm(x, y):
    return x*y // gcd(x, y)
```

## 3 개 이상 수들의 gcd, lcm 구하는 법

단순하게 생각하면 된다. 숫자 a, b, c, d 의 gcd 를 구한다고 해보자. 먼저 a, b 의 gcd 를 구하고, 그 결과와 c 의 gcd 를 구하고, 또 그 결과와 d 의 gcd 를 구하면 된다. lcm 도 같은 방식으로 구할 수 있다.

아래는 위 방식을 적용해본 코드다. 여러 수를 담은 `nums` 배열을 생성한 뒤, `x` 에 `nums` 의 첫번째 요소를 대입하고, 두번째 요소부터 `y` 로 순회하면서 gcd, lcm 을 계속 구해나간다.

```py
# ... 함수 정의 생략

nums = [75, 255, 150, 45]

x = nums[0]
for y in nums[1:]:
    x = gcd(x, y)
print(x)    # 15

x = nums[0]
for y in nums[1:]:
    x = lcm(x, y)
print(x)    # 7650
```
