## 문제

[프로그래머스, 모의고사](https://programmers.co.kr/learn/courses/30/lessons/42840?language=python3)

수학포기자들의 찍기 패턴과 답안을 비교하여 정답개수를 파악하는 코드의 구현이 핵심인 문제다.

수포자들의 이름, 찍기패턴, 정답개수 딕셔너리를 담은 리스트를 생성한뒤, 정답개수를 구해 대입하고, 문제에서 요구하는대로 리턴하도록 풀어보았다.

## 풀이

```python
from itertools import cycle

def solution(answers):
    # 1. [{'n': 수포자번호, 'p': 찍기패턴}, ... ] 형태를 담은 supo 리스트 생성
    supo = [
        {'n': 1, 'p': [1, 2, 3, 4, 5]},
        {'n': 2, 'p': [2, 1, 2, 3, 2, 4, 2, 5]},
        {'n': 3, 'p': [3, 3, 1, 1, 2, 2, 4, 4, 5, 5]},
    ]
    
    # 2. supo 반복, answers 와 패턴 매칭, 정답수 a 를 수포자별로 삽입
    for s in supo:
        s['a'] = sum(1 for x, y in zip(cycle(s['p']), answers) if x == y)
    
    # 3. 가장 높은 정답수 찾고, 해당 정답수인 수포자 이름 리스트 정렬하여 리턴
    m = max(e['a'] for e in supo)
    return sorted(e['n'] for e in supo if e['a'] == m)
```

`# 2` 에서 수포자들과 답안 매칭할 땐 `itertools` 모듈의 `cycle` 함수와 `zip` 내장함수를 이용하여 간단히 구할 수 있었다. 개인적으로 느끼기에 python 의 내장함수는 아주 잘 구축되어 있는 것 같다.
