## 문제

[프로그래머스, 프린터](https://programmers.co.kr/learn/courses/30/lessons/42587?language=python3)

아래처럼 문제에서 제시한 프린터 작동 방식대로 코딩을 하면 해결할 수 있는 문제다.

- 1: 인쇄 대기목록의 가장 앞에 있는 문서(J)를 대기목록에서 꺼냅니다.
- 2: 나머지 인쇄 대기목록에서 J보다 중요도가 높은 문서가 한 개라도 존재하면 J를 대기목록의 가장 마지막에 넣습니다.
- 3: 그렇지 않으면 J를 인쇄합니다.

## 풀이

```python
def solution(priorities, location):
    # 1
    watings = [{'p': p, 'i': i} for i, p in enumerate(priorities)]
    
    # 2
    a = 0
    while watings:
        j = watings.pop(0)
        if watings and any(j['p'] < x['p'] for x in watings):
            watings.append(j)
        else:
            a += 1
            if j['i'] == location:
                break
    
    # 3
    return a
```

`# 1` 에서는 대기문서를 `{'p': 중요도, 'i': 위치}` 형태로 `watings` 리스트를 생성하였다.

`# 2` 에서 `a` 는 프린트 횟수를 나타낸다. 반복문을 돌며 위에서 언급한 규칙대로 대기문서를 프린트하다가, `location` 에 해당하는 문서를 프린터했다면, 반복문을 탈출한다.
