## 문제

[프로그래머스, 위장](https://programmers.co.kr/learn/courses/30/lessons/42578?language=python3)

의상 종류별로 아이템 개수를 세서 총 경우의 수를 구하는 것이 핵심인 문제다.

예를들어 A 종류에 a1, a2, a3, B 종류에 b1, b2 가 있었다면 총 경우의 수는 11 이다. 해당 종류의 의상을 안입는 경우도 포함해야 하므로 A, B 종류에 None 이라는 가상의 아이템을 넣는다. 그러면 일단 총 4 * 3 = 12 의 경우의 수를 구할 수 있다. 그리고 최소한 한개의 의상은 걸친다는 전제조건이 있기에 None 끼리만 결합한 경우를 차감해야 하므로 결국에는 11 이 되는 셈이다.

## 풀이

```python
from collections import Counter
from math import prod

def solution(clothes):
    # 1.
    c = Counter([x[1] for x in clothes])
    
    # 2.
    return prod(x+1 for x in c.values()) - 1
```

`# 1` 에서 `Counter` 클래스를 사용하여 `clothes` 리스트의 두번째 요소를 모은다. 이렇게하면 의상종류 별로 아이템이 몇개씩 있는지 알 수 있다.

`# 2` 에서 각 의상종류의 아이템 개수를 계속 곱하기 위해 `math` 모듈의 `prod` 함수를 사용하였다. 그리고 위에서 언급한대로 None 인 경우도 고려해야하므로 (아이템 수 + 1) 을 계속 곱한뒤, 그 수에서 1 을 차감한다.
