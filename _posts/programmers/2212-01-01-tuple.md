## 문제

[프로그래머스, 튜플](https://programmers.co.kr/learn/courses/30/lessons/64065?language=python3)

주어진 문자열을 파싱하여 적절한 처리가 가능한 자료구조로 만드는 것, 그리고 자료구조를 바탕으로 결과 리스트를 만들어내는 것 두가지를 구현해야 하는 문제다.

## 풀이

```python
def solution(s):
    # 1
    t = s.lstrip('{').rstrip('}').split('},{')
    t = [[int(x) for x in y.split(',')] for y in t]
    
    # 2
    a = []
    for arr in sorted(t, key=len):
        a.extend([x for x in arr if x not in a])
    return a
```

`# 1` 코드는 문자열을 파싱하여 `[[숫자, ... ], ... ]` 형태의 이차원리스트를 만든다. `replace` 와 `split` 함수로 해결하였다.

`# 2` 코드에서는 `# 1` 에서 생성한 `t` 리스트를 길이 순대로 정렬한 뒤 반복한다. `a` 리스트에 없는 숫자만을 채우는 식으로 해결하였다.

## 참고

python 에는 `eval` 이라는 함수가 있다. 문자열 자체를 python 코드로 해석하여 처리해주는 함수다. `# 1` 코드를 아래처럼 수정해도 문제를 통과할 수 있다.

```python
# ... 생략

    # 1
    t = eval(s.replace('{', '[').replace('}', ']'))
    
# ... 생략
```

`{}` 기호를 `[]` 기호로 바꾸면 이차원리스트와 같은 형태가 되며, `eval` 함수에 적용하는 것만으로 실제 2 차원 리스트 대입과 동일하게 처리된다.

## 수정

다른풀이를 보다보니 정말 기막힌 방법으로 해결한 코드도 볼 수 있었다. collections 모듈의 `Counter` 클래스를 이용하는 것인데, 애초에 숫자만 골라내서 모두 `Counter` 클래스에 집어넣는 방법이다.

`Counter` 클래스의 특성 상 자연스럽게 가장 앞에 놓일 숫자의 개수가 가장 높게 나올 것이므로, 정렬만 해서 최종리턴하면 되는 셈이다.

아래는 `Counter` 사용 아이디어만 채용해서 나름대로 다시 작성해본 코드다.

```python
import re
from collections import Counter

def solution(s):
    t = Counter(int(x) for x in re.findall(r'\d+', s))
    return sorted(t, key=lambda x: -t[x])
```

정규식으로 숫자만 고르는 comprehension 표현식을 Counter 클래스 자료형을 적용한 뒤, key 에 매칭되는 value 값 내림차순으로 정렬만해서 최종리턴하는 구조다.
