## 문제

[프로그래머스, 예상 대진표](https://programmers.co.kr/learn/courses/30/lessons/12985?language=python3)

두 참가자가 계속 이겨나가면 몇번째 라운드에서 맞붙게 되는지를 파악해보라는 문제다.

라운드별 경기를 나타내는 리스트를 준비한 뒤, 각 참가자 위치를 표시하고, 라운드 진행 횟수에 따라 리스트를 줄여나가면서, 두 참가자가 맞붙게 되었을 때의 라운드 횟수를 리턴하는 방식으로 코딩하였다.

## 풀이

```py
def solution(n,a,b):
    # 1. 대진표 나타내는 g 리스트 생성, a, b 위치에 # 표시
    g = ['']*n
    g[a-1] = g[b-1] = '#'
    
    # 2. A, B 끼리 만나는 '##' 나올때까지 반복, 게임 라운드 r 증가
    r = 0
    while '##' not in g:
        r += 1
        g = [x+y for x, y in zip(g[::2], g[1::2])]
        
    # 3. r 리턴
    return r    
```

`# 2` 를 보면, 반복문을 돌면서, `zip` 함수를 사용하여 `g` 크기를 줄여나간다. 줄일 때 맞붙게 되는 사람끼리 붙이는데, 참가자 A, B 를 제외한 다른이들은 모두 빈칸이므로, `##` 이 되었을 때 A, B 가 맞붙는 것으로 볼 수 있다.
