## 문제

[프로그래머스, 괄호 변환](https://programmers.co.kr/learn/courses/30/lessons/60058?language=python3)

아래와 같이, 문제에서 제시한 규칙이 적용되는 재귀함수를 만들어 내는 것이 핵심인 문제다.

- 1: 입력이 빈 문자열인 경우, 빈 문자열을 반환합니다. 
- 2: 문자열 w를 두 "균형잡힌 괄호 문자열" u, v로 분리합니다. 단, u는 "균형잡힌 괄호 문자열"로 더 이상 분리할 수 없어야 하며, v는 빈 문자열이 될 수 있습니다. 
- 3: 문자열 u가 "올바른 괄호 문자열" 이라면 문자열 v에 대해 1단계부터 다시 수행합니다. 
- &#8195;&#8195;3-1: 수행한 결과 문자열을 u에 이어 붙인 후 반환합니다. 
- 4: 문자열 u가 "올바른 괄호 문자열"이 아니라면 아래 과정을 수행합니다. 
- &#8195;&#8195;4-1: 빈 문자열에 첫 번째 문자로 '('를 붙입니다. 
- &#8195;&#8195;4-2: 문자열 v에 대해 1단계부터 재귀적으로 수행한 결과 문자열을 이어 붙입니다. 
- &#8195;&#8195;4-3: ')'를 다시 붙입니다. 
- &#8195;&#8195;4-4: u의 첫 번째와 마지막 문자를 제거하고, 나머지 문자열의 괄호 방향을 뒤집어서 뒤에 붙입니다. 
- &#8195;&#8195;4-5: 생성된 문자열을 반환합니다.

## 풀이

```python
def solution(p):
    # 1
    def fn(w):
        # 1-1
        if w == '':
            return ''
        
        # 1-2
        tmp, u, v, is_valid = 0, '', '', True
        for i, x in enumerate(w):
            tmp += 1 if x == '(' else -1
            if tmp < 0:
                is_valid = False
            if tmp == 0:
                u, v = w[:i+1], w[i+1:]
                break
                
        # 1-3
        if is_valid:
            return u + fn(v)
        else:
            return '(' + fn(v) + ')' + ''.join('(' if x == ')' else ')' for x in u[1:-1])
        
    # 2
    return fn(p)
```

`# 1` 부분은 문제에서 요구하는 `fn` 재귀함수로, `#1-1` 은 위 규칙의 1 번에 해당하는 부분이다. `#1-2` 는 위 규칙의 2 번에 해당하는 부분이다. `u`, `v` 는 문제에서 얘기하는 u, v 그대로이고, `tmp` 는 균형잡힌 괄호인지 탐색을 위한 보조 변수, `is_valid` 는 올바른 괄호인지 아닌지를 판별하는 변수다.

일단 올바른 괄호로 보고 반복문을 시작한다. `tmp` 는 여는 괄호가 등장할 때마다 1 을, 닫는 괄호가 등장할 때마다 -1 을 계속 더해가는데, `tmp` 가 0 이 되는 순간의 인덱스 `i` 까지는 균형잡힌 괄호가 된다. 또한 반복 중간에 `tmp` 가 음수가 되는 순간이 오면, 이는 닫는 괄호가 더 많이 나왔다는 것이므로 올바른 괄호가 될 수 없기에 `is_valid` 를 False 값으로 바꾼다.

`# 1-3`은 위 규칙의 3 과 4 를 구현한 코드다. `is_valid` 결과에 따라 요구하는 대로 재귀호출을 수행한다.

`# 2` 에서는 재귀호출의 결과를 최종리턴한다.
