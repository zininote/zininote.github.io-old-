## 문제

[프로그래머스, 2016년](https://programmers.co.kr/learn/courses/30/lessons/12901?language=python3)

a 월 b 일이 1 월 1 일 로부터 며칠만큼 떨어져있는지를 알면 해결할 수 있는 문제다.

## 풀이

```py
def solution(a, b):
    # 1. 요일을 나타내는 week 리스트, 매월 마지막날짜를 나타내는 eom 리스트 생성
    week = ['FRI', 'SAT', 'SUN', 'MON', 'TUE', 'WED', 'THU']
    eom = [None, 31, 29, 31, 30, 31, 30, 31, 31, 30, 31, 30, 31]
    
    # 2. a월 b일과 1월 1일의 차이 계산, 요일 리턴
    diff = sum(eom[1:a]) + b - 1
    return week[diff%7]
```

`# 1` 부분에서는 리스트 2 개를 생성했다. `week` 리스트는 요일을 나타내는데 금요일부터 시작한다. 문제에서 1 월 1 일은 금요일이라고 했기 때문이다. 그리고 `eom` 은 문제에서 윤년이라고 하였으므로 2 월을 나타내는 인덱스의 요소는 29 가 된다.

## 참고

사실 대부분의 프로그래밍 언어에는 날짜와 시간의 편리한 처리를 위한 모듈이나 함수가 제공된다. python 에서도 datetime 모듈 안에 있는 함수 등을 사용하여 아래처럼 이 문제를 풀 수 있다. 물론 이렇게 풀라는 의도는 아니겠지만 말이다...

```py
from datetime import date

def solution(a, b):
    return ['MON', 'TUE', 'WED', 'THU', 'FRI', 'SAT', 'SUN'][date(2016, a, b).weekday()]
```
