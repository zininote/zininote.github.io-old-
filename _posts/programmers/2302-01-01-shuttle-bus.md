## 문제

[프로그래머스, 셔틀버스](https://programmers.co.kr/learn/courses/30/lessons/17678?language=python3)

버스시간과 사람들의 도착시간을 비교하는 것과, 최소한 언제 도착해야 셔틀버스를 탈 수 있는지를 찾아내는 문제다.

문제를 풀고보니 자료구조를 어떻게 만들것인가를 가장 고민해야 할 문제로 느껴졌다. 또한 문자열로 주어지는 시간 끼리의 계산이나 비교도 해결해야했다.

셔틀버스의 좌석 하나가 하나의 요소가 되는 리스트를 만들어보았다. `[{'bus': 해당 버스 출발 시간, 'seat': 해당 버스의 좌석에 앉을 사람}, ... ]` 식으로 총 좌석을 만들었다. 리스트 안 딕셔너리 총 개수는 `버스운행횟수 * 버스좌석수` 가 된다.

이렇게 만들고 나면, 리스트의 제일 마지막 요소가 어떤 상태인지에 따라 정답 리턴이 가능하다. 제일 마지막 요소는 제일 마지막 버스의 제일 마지막 좌석이 되는데, 이 좌석이 비어있다면 해당 버스 운행시간에만 도착하면 된다. 안 비어 있다면 이 좌석에 앉은 사람보다 1 분만 빨리오면 된다.

그리고 시간끼리의 계산과 비교를 위해서, 해당 시간을 모두 정수도 만든 뒤, 정수끼리 계산이나 비교를 한다. 나중에 필요할 때만 다시 시간으로 돌리는 식으로 구현했다. 이렇게 하면 시간끼리의 계산이나 비교를 굳이 힘들게 구현할 필요가 없다.

## 풀이

```py
# 1
def t_to_i(t):
    return int(t[:2])*60 + int(t[3:])

# 2
def i_to_t(i):
    q, m = divmod(i, 60)
    _, h = divmod(q, 24)
    return str(h).zfill(2) + ':' + str(m).zfill(2)

def solution(n, t, m, timetable):
    # 3
    tt = sorted(map(t_to_i, timetable))
    seats = [{'bus': t_to_i('09:00')+t*x, 'seat': -1} for x in range(n) for _ in range(m)]
    
    # 4
    for x in seats:
        if tt:
            if x['bus'] >= tt[0]:
                x['seat'] = tt.pop(0)
        else:
            break

    # 5
    if seats[-1]['seat'] == -1:
        return i_to_t(seats[-1]['bus'])
    else:
        return i_to_t(seats[-1]['seat']-1)
```

`# 1` 과 `# 2` 는 정수를 시간으로, 시간을 정수로 바꿔주는 함수다.

`# 3` 은 `timetable` 에 있는 사람들 도착시간을 정수로 바꿔, 오름차순으로 정렬을 하여 `tt` 에 할당한다. 그리고 위에서 설명한 버스 좌석들을 나타내는 `seats` 리스트를 생성한다. 사람들이 앉기 전에는 빈자리이므로 -1 로 초기화하였다.

`# 4` 에서는 본격적으로 `seats` 리스트의 요소마다 `tt` 에 있는 사람들을 앉히는 작업을 한다. 모든 `seats` 요소를 반복하면서, `tt` 에 남은 사람이 있으면서, 또한 버스시간보다 빨리 도착해있다면, 자리에 앉힌다.

`# 5` 에서는 위에서 언급한대로 `seats` 제일 마지막 요소 좌석이 비어있는지 혹은 누가앉아있는지 여부에 따라 다르게 최종리턴한다.
