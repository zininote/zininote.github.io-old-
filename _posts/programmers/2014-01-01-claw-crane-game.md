## 문제

[프로그래머스, 크레인 인형뽑기 게임](https://programmers.co.kr/learn/courses/30/lessons/64061?language=python3)

크레인 동작을 구현하는 것이 핵심인 문제다.

동작 구현을 좀 쉽게 하는 방법을 궁리하다가, 게임보드 모양을 바꿔보기로 했다.

주어지는 게임보드 형태가 크레인이 위에서 아래로 내려오는 것으로 구현되어있는데, 크레인이 오른쪽에서 왼쪽으로 움직이는 것에 맞도록 게임보드를 변경하고 시작한다. 그리고 0 요소는 제거를 했다. 이렇게해야 크레인 동작 코딩이 좀 쉬울 것 같았다.

## 풀이

```py
def solution(board, moves):
    # 1. board 행/열 전환하여 b 할당, 0 요소 제거 => # 3 코드 간소화 목적
    b = [e[::-1] for e in zip(*board)]
    b = [[e for e in row if e != 0] for row in b]
    
    # 2. 바구니 a, 사라진 인형수 c 초기화
    a, c = [], 0
    
    # 3. moves 반복, 인형있다면 바구니에 추가, 같은 인형이라면 제거 시행
    for m in moves:
        m -= 1
        if b[m]:
            tmp = b[m].pop()
            if a and a[-1] == tmp:
                a.pop()
                c += 2
            else:
                a.append(tmp)
                
    # 4. c 리턴
    return c
```

`# 1` 코드는 주어지는 `board` 를 재구성하는 부분이다. 행과 열을 바꾸고, 요소 순서를 뒤집으면 위에서 아래가 아닌 오른쪽에서 왼쪽으로 크레인 동작을 나타낼 수 있다.

`# 3` 코드의 바구니 처리는, 반복순회하는 m 라인에 인형이 있다면 `pop` 메서드로 뽑아내고, 바구니 마지막에 이미 같은 인형이 있는지 보고, 있다면 바구니의 인형도 `pop` 메서드로 제거하고, 두개의 인형이 제거되므로 `c` 변수를 2 증가시킨다. 아니라면 바구니에 인형을 추가한다.

사전에 `board` 리스트를 조작하기 쉽도록 변형했기 때문에, 크레인 동작을 나타내는 `# 3` 코드가 생각보다 간단하게 구현된 것 같다.
