## 문제

[프로그래머스, 다트 게임](https://programmers.co.kr/learn/courses/30/lessons/17682)

인수로 전달되는 `dartResult` 문자열을 파싱하는 것이 가장 중요한 문제로 보였다.

문자열을 파싱, 세트별로 [[점수, 보너스, 옵션], ... ] 형태로 2차원 배열을 만들어서, 이를 순회, 세트별로 점수를 계산한 다음, 그 합계를 제출하는 방식으로 풀어보기로 했다.

`dartResult` 파싱에 정규식을 사용하였다. 문제를 푸는 과정에서 뜻하지 않게 정규식 학습을 많이 하였다. 장시간의 삽질 끝에 "문자열" 을 2차원배열 형태로 바꿀 수 있었다.

## 풀이

```js
function solution(dartResult) {
    // dartResult 파싱
    // [[점수, 보너스, 옵션], ... ] 형태로 변환, d 에 할당
    var d = dartResult.match(/\d+[SDT][*#]?/g);
    for(var [i, x] of d.entries()) {
        var tmp = x.split(/(\d+)([SDT])([*#]?)/);
        d[i] = tmp.slice(1, -1);
    }
    
    // 보너스와 옵션 조건을 bonus, op 에 할당
    var bonus = {'S': 1, 'D': 2, 'T': 3};
    var op = {'*': 2, '#': -1};
    
    // 세트별 점수 계산, scores 에 할당
    var scores = [];
    for(var [i, x] of d.entries()) {
        scores[i] = (+x[0])**bonus[x[1]] * (x[2] ? op[x[2]] : 1);
        if(i && x[2] == '*') { scores[i-1] *= op[x[2]] };
    }
    
    // 세트별 점수합계 리턴
    return scores.reduce((a, b) => a+b, 0);
}
```

```py
import re

def solution(dartResult):
    # dartResult 파싱
    # [[점수, 보너스, 옵션], ... ] 형태로 변환, d 에 할당
    d = re.findall(r'(\d+)([SDT])([*#]?)', dartResult)
    
    # 보너스와 옵션 조건을 bonus, op 에 할당
    bonus = {'S': 1, 'D': 2, 'T': 3}
    op = {'*': 2, '#': -1}
    
    # 세트별 점수 계산, scores 에 할당
    scores = [0]*3
    for i, x in enumerate(d):
        scores[i] = int(x[0])**bonus[x[1]] * (op[x[2]] if x[2] else 1)
        if i and x[2] == '*': scores[i-1] *= op[x[2]]
            
    # 세트별 점수합계 리턴
    return sum(scores)
```

정규식 사용은 Python 이 훨씬 편했다. `findall` 함수 한번만으로 원하는 형태를 금새 만들 수 있었다.

Javascript 에 비슷한 정규식 표현(추가로 나누고 싶은 부분에 괄호 적용)을 적용해봤는데 작동하지는 않았다. 어쩔 수 없이, 세트별로만 나눈 뒤, `split` 함수로 다시금 처리를 해줘야 했다.

마지막 리턴 부분도, Python 은 `sum` 함수를 빌트인으로 기본 제공하기에, 가독성 및 편리성은 더 뛰어난 듯 하다.
