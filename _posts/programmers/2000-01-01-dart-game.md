## 문제

[프로그래머스, 다트 게임](https://programmers.co.kr/learn/courses/30/lessons/17682?language=python3)

문제를 보면 해결해야 할 것은 크게 두가지로 보인다. `dartResult` 문자열을 파싱하는 것과 세트별 점수 계산이다. 특히 문자열 분석에는 정규식을 사용하기로 했다.

## 풀이

```py
import re

def solution(dartResult):
    # 1. dartResult 를 [(점수, 보너스, 옵션), ... ] 형태로 파싱, dart 에 할당
    dart = re.findall(r'(\d+)([SDT])([*#]?)', dartResult)
    
    # 2. 보너스와 옵션 조건을 담은 bonus, op 딕셔너리 생성
    bonus = {'S': 1, 'D': 2, 'T': 3}
    op = {'*': 2, '#': -1}
    
    # 3. 세트별 점수 담을 a 리스트 생성, dart 반복하며 점수계산 후 a 에 삽입
    a = [0]*3
    for i, (s, b, o) in enumerate(dart):
        a[i] = int(s)**bonus[b] * (op[o] if o in op else 1)
        if i and o == '*':
            a[i-1] *= op[o]
    
    # 4. a 합계 리턴
    return sum(a)
```

`# 1` 코드에서 사용한 정규식은 프로그래밍 언어마다 사용방식에 조금씩 차이가 있다. [python 공식문서](https://docs.python.org/3/howto/regex.html)를 참고해서 경험치를 많이 쌓아두는 것이 좋을 것 같다.
