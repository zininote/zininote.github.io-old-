## 문제

엑셀 sheet 이름을 엑셀문서 내부에서 사용하고 싶을 때가 있다. 다만 수동으로 일일이 키인하는 것이 아니라, sheet 이름을 바꿨을 때 sheet 이름이 사용된 다른 모든 곳도 자동으로 바뀌기를 바랄 수 있다.

불가능한 것이 아니다. **엑셀 sheet 이름을 함수식으로 가져오는 방법**만 알면 된다. 함수이기 때문에 참조값이 바뀌면 결과도 자동으로 재계산된다. 아래부터 이 함수식을 소개하고자 한다.

## 함수식

![그림00](https://zininote.github.io/img/excel/excel-0002-01-01-00.png)
{:.center}

```excel
= MID( CELL( "filename", 셀 ), FIND( "]", CELL( "filename", 셀 )) + 1, 256 )
```

위 함수식에서 `셀` 부분에는 이름을 가져오길 원하는 sheet 의 아무셀이나 지정하면 된다.

주의해야 할 점이 있다. **반드시 파일이 저장된 상태에서 사용해야 제대로 작동**한다. 예를들어 새로 엑셀 문서를 만들어서 아직 한번도 저장도 안한 상태에서 위 함수를 사용하면 에러를 발생시킨다. 만일 에러가 발생했다면 일단 저장부터 하고 F9 키를 눌러 재계산을 해줘야 한다.

CELL 함수 때문인데, 이 함수의 첫번째 인수를 "filename" 이라 지정하면 폴더경로, 파일이름, Sheet 이름 등을 모두 포함한 셀의 정보를 반환한다. 따라서 아직 한번이라도 저장을 안한 갓 만든 엑셀 문서라면 파일정보가 있을 리 없으므로 에러가 발생하는 것이다.

## 참고

같은 기능을 하는 사용자함수를 만들 수 있다.

VBA Editor(ALT+F11) 를 열어서 "삽입" > "모듈" 메뉴를 선택하면 나오는 창에, 아래 코드를 넣기만 한 뒤, VBA Editor 를 닫고 다시 엑셀로 돌아오면 사용할 준비가 된다.

```vb
Function SheetName(rCell As Range) As String
    Application.Volatile
    SheetName = rCell.Parent.Name
End Function
```

```excel
= SheetName( 셀 )
```

엑셀에서 위처럼 사용하면 되며, `셀` 부분에는 이름을 얻고자 하는 곳 아무셀이나 지정하면 된다.

참고로 위 VBA 사용자함수는 CELL 을 사용한 함수식과는 다르게 파일로 저장하지 않은 상태에서도 사용이 가능하다.
